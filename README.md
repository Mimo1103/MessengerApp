# Messenger App
## Overview
A messaging application built in Java that allows users to communicate via a locally hosted REST API

## How to run
1. Launch the server
2. Start the client
3. Start messaging once the client is connected to the server

## Features
- Client-server communication with database integration
- Sending and receiving messages
- JSON-based data exchange for messages

## My contribution
- Implemented a client-server architecture using Spring Boot and Java
- Developed REST API endpoints for communication
- Implemented JSON-based data exchange
- Built the graphical user interface with JavaFX
- Integrated client and server components into a working messaging system

## Technologies
- Java
- Spring Boot
- JavaFX
- Jackson

## Architecture
The application consists of a locally hosted server to which the client connects:
- The server is built with Spring Boot and handles all incoming messages via a REST API
- The client uses JavaFX to provide a graphical user interface for sending messages
